# TCP/IP

## IP

### 用語

- IPv4
  - 32bitのアドレス体系
  - 192.168.1.1のようなドット区切り10進数表記
  - `/24`(256個)が範囲指定として多い
  - 約43億個のアドレスが世界中で上限 → すでに枯渇
- IPv6
  - 128bitのアドレス体系
  - IPv6の表記ルールは128bitを16bitずつ8グループに分けてコロン区切り16進数表記
  - `/48`や`/56`がISP範囲指定の割り当て単位
  - アドレス数上限は約340澗(かん)個 → 事実上枯渇しない
  - `240a::/16`はNTTコミュニケーションズに割り当てられた範囲で、日本のIPv6アドレス

```text
240a : aaaa : bbbb : cccc : 1111 : 2222 : 3333 : xxxx
 ①     ②     ③     ④      ⑤      ⑥      ⑦      ⑧
```

- ISP: インターネットサービスプロバイダ
- CGN(キャリアグレードNAT)
  - CGNは複数の出口IPを持ち、宛先サービスごとに異なる出口IPが使われることがある
  - IPv4の43億個では世界中のデバイスに対応できなくなったため、CGNはその場しのぎの延命策。IPv6への移行が本筋

- デュアルスタック
  - IPv6優先
  - ISP,ルーター、端末(mac対応)の3つが全て揃ってデュアルスタック対応
